# Лабораторная работа №2   

## Введение
Этот репозиторий содержит код, эмулируюзий поведение простых shell команд

## Как сделано
- Используется typer, инициализация комманд, их использование и вывод результата производистся используя эту библтотеку
- Исходный код для каждой из команд лежит в `src/commands`
- Тесты написаны используя pytest
- Команды, удачное их выполнение и ошибки логгируются в `shell.log`
## Как использовать
- Запуск через `python -m src.main` или через IDE
- Запуск тестов, введя `pytest tests/tests_easy.py -v` из корня
- Запустив программу, вы можете вводить команды из списка ниже
- Чтобы выйти, введите quit
## Список доступных команд
### Все они выдадут ошибку если нет прав доступа, или файла/директории по указанному пути нет
### 1. `cat`: отображение содердимого файла переданного по path 
   - Пример использования: `cat meow.txt`
### 2. `cd`: смена рабочей директории по переданному path
   - Пример использования: 
     - `C:\Users\User\PycharmProjects\lab2-python-mai\src $ cd commands`
     - `C:\Users\User\PycharmProjects\lab2-python-mai\src\commands $ `
   - Поддерживает относительные и абсолютные пути
   - Поддерживает домашнюю директорию `~`
### 3. `cp`: копирует файл или директорию (по флагу `-r`) в существующую или новую директорию
   - Пример использования: `cp a.txt letters`
     - теперь существует 'letters/a.txt', и `a.txt` в своем изначальном расположении
### 4. `ls`: отобрадает список содержащихся файлов и директорий в каталоге. По флагу `-l` подробно
   - Пример использования: `ls letters`
     - Вывод:
     - `a.txt`
     - `b.txt`
     - ...
     - `z.txt`
### 5. `mv`: позволяет переименовать или переместить файл/директорию (по флагу `-r`)
   - Пример использования: `mv letters betters`
     - `letters` теперь называется `betters`
     - `mv letters src/book` -> `letters` перемещено в `src/book`
### 6. `rm`: позволяет удалить файл/директорию (по флагу `-r`), просит подтверждение у пользователя
     - Пример использования: `rm letters -r`
       - `are you sure? Y/n`
       - При вводе `Y` директория `letters` удалена
